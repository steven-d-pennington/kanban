## Technology Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS, Zustand
- **Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **Agents**: Node.js, Anthropic Claude API
- **Deployment**: Vercel
